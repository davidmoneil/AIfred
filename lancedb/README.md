# LanceDB

**Purpose**: Vector database for embeddings and RAG operations.

**Layer**: Soma (infrastructure)

---

## What Belongs Here

- Vector database files
- Embedding indices
- RAG data storage

## Usage

LanceDB supports:
- Semantic search
- Document embeddings
- RAG (Retrieval Augmented Generation)

## Note

This directory contains database files, not configuration.
Configuration lives in `/.claude/config/`.

## Structure

| Directory | Contents |
|-----------|----------|

- Self-reflection reports
- Maintenance cycle reports
- Evolution cycle reports
- R&D discovery reports
- Milestone review reports

## What Does NOT Belong Here

- Project development reports → `projects/project-aion/reports/`
- MCP validation logs → `logs/mcp-validation/`

## Key Distinction

**Operational reports** (here): Jarvis' self-improvement cycle outputs
**Project reports** (`projects/project-aion/reports/`): Development work results

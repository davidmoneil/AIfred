# Attribution & Credits

## Project Origins

### AIfred Baseline

Jarvis is built upon the **AIfred** framework created by **David O'Neil**.

- **Repository**: [AIfred](https://github.com/davidmoneil/AIfred)
- **License**: See AIfred repository for license terms
- **Baseline Commit**: `2ea4e8b` (as of Jarvis v1.9.5)

AIfred provides the foundational architecture including:
- Claude Code hook system
- Context management patterns
- Agent framework
- Setup wizard structure

### Project Aion

**Project Aion** is the development initiative for extending and customizing the AIfred baseline into Jarvis.

- **Author**: NS Cannon
- **Repository**: Jarvis (this repository)
- **License**: See repository LICENSE file

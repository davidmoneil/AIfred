# Workflows

**Purpose**: Repeatable procedures for completing large tasks within sessions.

**Layer**: Nous (procedural knowledge)

---

## What Belongs Here

- Step-by-step procedures
- Task completion workflows
- Session-level procedures

## What Does NOT Belong Here

- Behavioral patterns → `patterns/`
- Standards and rules → `standards/`
- Session work plans → `plans/`

## Hierarchy Context

```
Standards (MUST follow)
    ↓
Patterns (SHOULD follow)
    ↓
Workflows (Large task procedures) ← YOU ARE HERE
    ↓
Designs (Architecture philosophy)
    ↓
Plans (Session-level work)
```

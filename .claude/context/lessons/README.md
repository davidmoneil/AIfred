# Lessons

**Purpose**: Self-corrections, learnings, and accumulated wisdom from experience.

**Layer**: Mind (long-term memory)

---

## Structure

| File/Directory | Contents |
|----------------|----------|
| `corrections.md` | Documented corrections and their lessons |
| `self-corrections.md` | Self-identified mistakes and improvements |
| `index.md` | Navigation and categorization |
| `patterns/` | Recurring patterns identified |
| `problems/` | Problem descriptions and solutions |
| `solutions/` | Solution templates |

## What Belongs Here

- Corrections and their lessons
- Recurring mistake patterns
- Problem-solution pairs
- Accumulated wisdom

## Lifecycle

1. **Capture**: `self-correction-capture.js` hook prompts for lessons
2. **Document**: Add to appropriate file
3. **Categorize**: Update index
4. **Reference**: Consult during similar situations

## Key Principle

Lessons are **accumulated wisdom** that shapes future behavior.
They feed into AC-05 Self-Reflection cycles.

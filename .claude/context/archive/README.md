# Archive

**Purpose**: Historical session states and archived context documents.

**Layer**: Nous (historical memory)

---

## Structure

| Directory | Contents |
|-----------|----------|
| `session-state/` | Archived session state snapshots |

## What Belongs Here

- Historical session state files
- Superseded context documents
- Archived operational state

## Lifecycle

Files are archived here when:
- Session state is saved before major changes
- Documents are superseded by newer versions
- Historical reference value warrants preservation

# Sessions

Session summaries capturing accomplishments, decisions, and learnings.

## Naming Convention

Files are named: `YYYY-MM-DD-<slug>.md`

## Capture

```bash
/capture session "Completed M3 JICM complement commands"
```

## Template Contents

- Accomplishments
- Decisions made
- Learnings
- Files modified
- Commits
- Next steps

## Browse

```bash
/history recent 5
/history show sessions/2026-01-23-jicm-commands.md
```

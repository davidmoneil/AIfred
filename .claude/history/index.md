## Usage

```bash
# Search all entries
/history search "keyword"

# Show recent entries
/history recent 10

# Browse by category
/history category learnings/patterns

# Browse by tag
/history tags #archon

# View statistics
/history stats

# Promote to Memory MCP
/history promote learnings/archon/2026-01-22-example.md
```

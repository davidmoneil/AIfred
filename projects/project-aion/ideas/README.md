# Ideas

**Purpose**: Exploratory notes and brainstorming — not yet committed to implementation.

---

## Structure

| Directory | Contents |
|-----------|----------|
| `current/` | Active exploration and consideration |
| `archive/` | Explored and decided (adopted or rejected) |

## What Belongs Here

- Brainstorm documents
- Exploratory research notes
- "What if" proposals
- Feature concepts before commitment

## What Does NOT Belong Here

- Committed designs → `designs/`
- Implementation plans → `plans/`
- Execution results → `reports/`

## Lifecycle

1. **Explore**: Idea starts in `current/`
2. **Decide**: Evaluate feasibility and value
3. **Commit or Archive**:
   - If adopting → Create design in `designs/current/`, archive idea
   - If rejecting → Archive with rationale
